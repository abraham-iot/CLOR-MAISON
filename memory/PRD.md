# CLOR MAISON - PRD (Product Requirements Document)

## Original Problem Statement
Página web para negocio de uñas CLOR MAISON. Debe ser muy atractiva y moderna, mostrar galería de trabajos de uñas con imágenes reales, tener sección de contacto con WhatsApp, y un logo/eslogan destacado.

## User Personas
- **Clientas potenciales**: Mujeres interesadas en servicios de manicure/uñas profesionales en la zona de San Juan, Puebla

## Core Requirements (Static)
- Landing page moderna y elegante
- Logo/marca CLOR MAISON prominente
- Galería de trabajos de uñas (imágenes reales)
- Información de contacto con WhatsApp
- Diseño responsive (móvil y escritorio)

- **Colors**: Alabaster (#FAF9F6), Charcoal (#292524), Gold (#A67C52)
- **Style**: Minimalista, elegante, tipo revista Kinfolk
